# NoteNest Backend

Backend scaffold for the NoteNest collaborative knowledge base.

---

## Current State

- Express server
- Health endpoint
- No database
- No authentication

---

## Planned Features

- Notes API
- Search & indexing
- Role-based access
- Versioning

---

## Run Locally

```bash
npm install
npm run dev
```
Health check:
http://localhost:5001/health


---
# NOTICE

After cloning, always run 
```bash
npm install
```
inside backend/ before starting the server.
